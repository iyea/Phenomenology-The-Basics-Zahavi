# 第二章，胡塞尔的先验哲学转向：悬搁、还原和先验唯心论

胡塞尔注意到了纯粹描述性现象学的不充分性，他在《纯粹现象学和现象学哲学的观念》（1913）中转向了先验现象学。本章的任务就是说明胡塞尔的此变化。此外，胡塞尔在 1913 年之后并没有停止对其分析的发展，但是我们不会对他所有的变化作出说明，我们主要考察《观念 1》，以及他的最后一本书《欧洲科学危机和先验现象学》。

## 第一节，无前提性

胡塞尔认为，现象学的任务是，将关于存在和实在的本性的哲学核心问题主题化并阐释清楚。但是，我们并不能简单的接受哪些刻画我们日常生活的认识论和形而上学的设定（最基本的，这是一种实在论设定，是我们对独立于心灵、经验和理论的实在之存在的隐藏信念，胡塞尔称之为自然态度），因为这样的研究不够彻底性，胡塞尔坚持认为把自然态度的合法性作为想当然的，在哲学上是不可接受的。那么我们该怎么研究存在和实在的本性呢？胡塞尔的回答非常具有欺骗性：我们应该将研究的注意力转向实在的被给予性或者说现象。《观念 1》的 24 节，胡塞尔阐述了现象学的“诸原则之原则”：我们应该让原初给予的直观作为全部知识的来源，这个来源不容许任何权威（包括现代科学）的质疑。

要转向被给予者，还需要很多方法论的准备：

1. 悬搁。我们要对自然态度的实在论倾向、对它的合法性加上括号（悬搁）。我们不是去否认自然态度，而是仅仅中立化对实在的教条态度。
2. 先验还原，它表示我们对主体和世界之间的关联进行主题化，从而找到自然态度的先验基础的过程。

显然，悬搁自然态度是进行先验还原的前提条件。胡塞尔把悬搁比作从二维到三维的转变，突然之间，那个恒久地、隐藏地运作着的先验主体性，被作为“显现之可能性”的主观条件，被揭示出来。

## 第二节，笛卡尔式的方法和本体论的方法

如何进行先验还原？胡塞尔提供了三种方法：笛卡尔式的、心理学的、本体论的。我们主要介绍其中的两个：

1. 笛卡尔式的。空间——时间性的被给予的对象，和意识被给予其自身的方式，有一个重要的区别，那就是前者都是有角度的、有限的侧面被给予的，而后者是直接在其整体性中显现的。因此，主体性与任何对象之间有决定性的差异。对于这两者的关系，存在一种主张叫做“主体性优先”。受笛卡尔的启发，胡塞尔想象无世界的主体是可能的，但是想象无主体的世界却是不可能的。站在现象学的显现视角上来说，世界只能向主体显现，主体却不需要世界才能存在。主体（内在性）的显现只依赖于其自身，因此是绝对的和自主的。这样，如果存在一个无世界的主体，那么仅仅把意识还原成世界的一部分，这样来看待意识就是不够的。这样，我们就克服了对于意识的自然主义，意识被揭示为脱离世界的领域（先验领域）。但是，我们必须还要看到，意识确实是世界的一部分。所以，意识既是经验的（作为实在的、世界的一部分显现），又是先验的（作为绝对的、作为世界显现的前提条件）。意识何以拥有这种二象性？这是一个很大的谜。总而言之，胡塞尔把经验自我和先验自我当成了对于同一个自我的两种理解，并没有像笛卡尔那样分裂开。面对主体的二象性，我们如何才能让更清晰地思考主体的先验成分？答案显然是通过悬搁，脱离自然态度。但是，笛卡尔式的先验还原当然是有问题的，因为它暗示了，现象学的任务是在与世界和主体间性孤立开、分离开的领域中，研究纯粹的主体性。
2. 本体论的。我们如果想要研究空间——时间性的被给予的对象，那么必须承认：这个现象必然是被我们表象、感知和判断的，只要我们想要认识对象，这对象就必须在与主体的相关中被理解。因此，我们就不能仅仅关注于被给予的对象，还要关注 让对象如其所显现的那样起作用的、与对象相关的 经验结构和意向性。这个主体也不是经验主体，而是作为经验的条件的先验主体，是显现本身的可能性条件。但是，如果我们只是沉浸于认识对象，而没有认识到我们如何认识对象的结构，那么我们就仍然沉溺在自然态度中。只有通过悬搁，我们才能将先验主体揭示出来。因此，通过本体论的先验还原，我们可以避免上述笛卡尔式的先验还原踩入的陷阱。因为我们的目标在于认识世界如何以其所是的方式被揭示，从而我们才必须关注到那个处于世界之外的先验主体。

现在，胡塞尔的先验转向现在就非常可理解了。如果我们希望真正理解对象是什么，我们就不得不转向经验这些对象的先验主体性。没有主体性就没有对象的显现（现象）。记住，胡塞尔现在采取先验哲学的立场：主体性是实在性的前提。

## 第三节，对现象学的误解

这里介绍一个最重要的关于现象学的深刻分歧。

弗莱斯戴尔、德赖富斯、Miller、史密斯和麦金泰尔（加尼福利亚学派，西海岸理解）为胡塞尔的意向性理论的弗雷格式的理解辩护。根据他们，意向对象（noema）必须与意向活动（noesis）和被意向的对象（object as it is intended）严格区分开。noema 是作为 noesis 和 object as it is intended 之间的中介而存在的。noesis 不是被 intended 的东西，而是 intend 的手段。我们依赖 noema 才能到达对外在 object 的指称。准确地说，noema 是“中介性的意向性实体”，noesis 通过 noema 来 intend 外部的 object as it is intended。

Sokolowski、德鲁蒙德、Hart、科布——史蒂文斯（东海岸理解）认为意向性是意识经验的根本特性。根据他们，在进行先验还原之后，我们仍然关注的是外部的 object as it is intended，noema 不能被理解为观念性的意义、概念或者命题，从而根据这些手段来通往 object as it is intended，毋宁说，在现象学反思的语境中，noema 就是 object as it is intended，这两者是彻底等价的。noema 就是被感知的对象、被回忆的片段、被命题化的那个现实事态。东海岸理解也承认，我们确实通过意义、概念和判断去认识 object as it is intended，但是这些东西本来就是 object as it is intended 最根本的组成部分，对象就是把这些显现出来从而被我们理解的。

可以看到，东海岸理解完全拒斥了表象主义，也拒绝把胡塞尔当作内在论者。Zahavi 支持东海岸理解。虽然描述现象学可以理解为一种意义理论，从而拒斥本体论，即《逻辑研究》中胡塞尔的立场，但是先验现象学不能，因为在《观念 1》中，胡塞尔指出，因为方法论的原因，现象学最终会将它最初打上括号（悬搁）的东西也包括进来。胡塞尔宣称，完全发展的现象学是真正的、实现了的本体论。所有的本体论概念和范畴都在它们与构成性的主体性的关联中被阐明。现象学拒斥朴素的形而上学（类似于自然态度），但是不拒斥形而上学本身。所有形而上学问题都会回归到现象学的基础上来，在那里，它们才找到了从直观而形成的、真正的先验形式和方法。认为现象学（胡塞尔）对实在或者存在问题完全不感兴趣，是对现象学目标的误解。

然而，形而上学这个词毕竟是多义的：

1. 处理第一原则的思辨性地构造出来的哲学系统。
2. 关于超越感性和现象的实体的学说。
3. 描述实在的客观主义的、拒斥任何视角的企图。
4. 回答“为什么存在而不是无”问题。
5. 建立在某种二元对立的、逻辑上的思维方式。
6. 回答“人类生活的意义”问题。
7. 对存在着的实在的本性的系统性反思。

我们这里谈论的是最后一种。

先验现象学和形而上学是两种非常不同的事业。站在先验的角度看，形而上学是前批判的和朴素的，它没有离开过自然态度，没有认识到思想的规定性。但是，我们不能说，因为现象学对形而上学采取中立化的态度，那么现象学就能接受所有形而上学。如此主张，就会让先验现象学变成现象学的心理学（区域性的本体论事业，只研究对任何可能的意向性而言，主体必须具有的先天结构）。但是，现象学的目的还不仅仅是这个。实际上，胡塞尔已经通过先验还原，超越了意义与存在之间的、客观主义的区分。当然，这不是说任何有意义的东西（意向内容）都存在。在现象学的定义里面，存在的对象指称的是能够被亲身地在直观中被给予的对象（能够被充实的意向内容）。

换句话说，胡塞尔的形而上学是完全绑定到先验主体身上的，不能脱离了对先验主体的理解来独立地理解形而上学。但是这也不代表胡塞尔认为“绑定到先验主体身上的形而上学”和“研究真实的存在的形而上学”是两个东西，从而把“绑定到先验主体身上的形而上学”相对主义化，认为它是心灵的印象，这是不对的。毋宁说，在胡塞尔那里，“研究真实的存在的形而上学”只能作为“绑定到先验主体身上的形而上学”来研究，并且“绑定到先验主体身上的形而上学”是全局唯一的、客观的、能够触及实在的、有立场的。

然而，如何摆脱“绑定到先验主体身上的形而上学”的主观性呢？事实上，在《观念 1》的最后一部分，即“理性现实”的那部分，胡塞尔处理了关于 客观实在 和 理性意识 的关联问题。正是这种分析把胡塞尔引导到了“先验主体间性是世间性的客观性与实在性的基础”。胡塞尔说“那个内在的第一存在、这个先于且承载每一世间性的客观性的存在，就是先验主体间性。这以不同方式实现就交流的单子宇宙”。

然而，西海岸理解（弗雷格）的理解也并非一无是处，而且也有文本证据。两者之间各有优势。比如：

1. 西海岸理解认为东海岸理解不能解释“幻觉”的问题。
2. 东海岸理解认为西海岸仍然沉浸在心灵主义和表象主义之中，而东海岸理解和世界的关系更近，从而可以贴近后期胡塞尔。

对意向对象的考察是最有趣的，但是作为导论，本书无法深入下去。

胡塞尔也经常被称呼为基础主义者。这是因为，作为先验哲学家，胡塞尔认为，对先验主体性的阐明，是使得所有其他科学变得可理解的、发生于其中的研究。但是，胡塞尔和传统的基础主义者仍然有区别：

1. 胡塞尔没有认为它的先验分析是结论性的、绝对的。它是可探索的领域，是仅能在无限的历史过程中被实现的理念（这类似于黑格尔）。
2. 胡塞尔不采取演绎的公理体系。胡塞尔不认为现象学可以指出某些公理，从这些公理中可以推导出实证科学的内容（这不类似于黑格尔）。

还应该注意到，胡塞尔作哲学的动机，不仅仅是理论性的先验条件的研究（类似于康德）；还是伦理的：对 处于绝对自我责任中的生活 的伦理追求。胡塞尔谈到基于对先验基础的现象学探究的、基于明证性的自我负责的生活。这类似于苏格拉底。在这样的具有纯粹理性的生活中，人们不懈地在自我反思和彻底的责任中，实行对 生活目标 的批判，以及对 实现生活目标的道路（手段）的批判。这种批判的方法是，考察 所有合法性和认识的原初来源，用现象学的术语，就是回归到清晰性、洞见和明证性。正是因为我们有这样的伦理动机，所以我们才能孜孜不倦地在无限的历史中追逐 先验分析 这个无限的理念。重要的不是对 绝对真理 的占有，而是对 生活在绝对自我责任中 的尝试。而且，胡塞尔也强调，个人的自我责任只有在于他人的关系中才能充分实现，这就又引入了主体间性。

## 第四节，胡塞尔的先验唯心论

在《逻辑研究》中，胡塞尔已经表明，只有唯心主义的认识论才是融贯的。这里的唯心主义指代的是，主张观念物不能被还原为心理或者物理的实体或者过程（逻辑的柏拉图主义）。在《笛卡尔的沉思》中也可以找到同样的命题。但是应该在远为深刻的意义上来看待它。胡塞尔甚至将现象学等同于先验唯心论，并且指出它与传统的唯心论不同，因为它不赞同把意向对象（东海岸理解）还原为（心灵中的）感性杂多。那我们又如何这种“承认存在超越主体的对象的唯心论”呢？

1. 在这个意义上，现象学是唯心主义的：实在物必须通过主体来表达它自身，实在物依赖于主体性，现象学拒绝对实在物的本体论进行客观主义的解释。
2. 在这个意义上，现象学是超越的：实在物总有不同于主体的预期（让主体感到惊奇）而显现的可能性。实在物在绝对的意义上独立于主体的视角，主体的视角不能通达。

总之，胡塞尔将先验唯心论描述为这样一种尝试：通过对主体的 构成性的意向性 的系统性分析，来理解和阐明世界的丰富性和超越性。因此，先验唯心论可以作为对曾经悬搁的自然态度的挽回。胡塞尔说道：超越的世界、人类的主体间性，这些没有被现象学反思所取消，也没有被贬值和变化，而仅仅是被理解了。

## 第五节，构成的概念

以上，我们提到先验现象学包含对 构成性的主体性、构成性的意向性 的研究。但是，这里所说的 构成性 究竟是什么呢？有几种理解：

1. 构成性是主体的创造性，是彻底的唯心主义（贝克莱）。
2. 构成性是主体和经验到的对象之间的认识论关系（现象学东海岸理解）。
3. 构成性是意义的维度，而不是存在的维度（现象学西海岸理解）。

胡塞尔本人的立场依然是含糊不清的。Zahavi 认为，对于构成性的理解应该是，它是让 被构成者（意向对象）按其所是地展现 的过程。构成并不是在制作和编造的意义，而是让实体在其客观性中被看到（海德格尔所说）。这个构成性的过程，不是先验自我从 无 之中故意地、单方面地推动它发生的。胡塞尔承认，显现的可能性条件，不只有主体性（自我），还有非主体性（非我）。胡塞尔说到“世界作为先验的非我”。因此，胡塞尔的先验成分中不只有主体性，还有非主体性，也包括先验主体间性，这些先验部分是相互纠缠的。这不是二元论，而是复杂的构成。构成性是在主体性和世界的结构中展开自身的过程。在这个意义上，胡塞尔的立场非常接近于梅洛庞蒂。（当然，也接近于黑格尔的“精神”。）

综上，构成性不能简单理解为 1、2、3 这种单纯的认识论理论。我们必须认识到，存在与意识是相互依赖，并且结合在先验主体性之中的。胡塞尔“单子”的概念不仅包含了意识的意向性，也包括了通过它构成的对象。可以看到，胡塞尔的主体性概念不断地扩大，甚至破坏了主客体的传统对立。实际上，胡塞尔运用两种主体性的概念，前者是狭义的、通常所用的意识维度；后者是包含意识和世界两者的主体性。完整的主体性是一个经验着世界的生命，胡塞尔用“世界——意识的生活”这样的概念形容。

在胡塞尔晚期的著作中，它放弃了构成者（意识）和被构成者（对象）的静态相关性，因为他发现，构成性的主体，就是在构成的过程中 被构成的。因此，构成性这个概念是反身性的。主体作为构成者，在构成的活动中，也自我构成着自己。（这里走向了黑格尔“精神的自我意识”。）因此，胡塞尔的晚期反思，否决了他在早期提出的“无世界的先验主体”存在的可能性。进一步地，在这个反思的过程中，先验主体也不能作为彻底脱离经验主体的独立存在。先验主体作为构成过程的一部分，必然要将自己自身构想为一个世界上的实体。更细致地探讨在此导论的第三部分：先验主体被肉身化和社会化，它才能构成一个客观的世界。（仍然参考黑格尔对“耶稣的肉身化”的解释。）

进一步地，上面提到了“社会化”，即，想要理解胡塞尔的最终立场，上述一直绕不开的概念就是主体间性，作为第三个必然的先验概念存在，它和主体、世界这两个先验概念的地位是并列的（主体——主体间性——世界）。胡塞尔也会提到，主体和世界的构成，是在主体间发生的。关于主体间性，他说“许多单子在自身之中构成客观世界，并且将自身在这个世界之中空间化和时间化，并且实现它们自身（尤其是作为人）”。世界的构成、自我的展开、主体间性的建立，是一个相互联系的自发过程的组成部分。正如胡塞尔在《观念 2》中写道的“我、我们、以及这个世界住在一起”。

胡塞尔最开始的立场仅仅是，实在因为主体而存在（现象学意义上的、作为显现的直观而存在）。但是最终，胡塞尔意识到：

1. 主体不能保持不被它的构成活动所影响。它被吸了进去。
2. 构成不仅仅是单个主体和世界的关系，而是主体间性的过程。

此后，胡塞尔面临的问题就是要澄清 主体——主体间性——世界 之间的关系。关键在于，这三者是互相联系的，提及其中任何一个都无法避免地走向另外两者。

胡塞尔的最终立场会使某些读者想起黑格尔的成分。但是，正如芬克指出的，现象学并不是思辨的构造，而是对现象学还原的基本洞见的表达。胡塞尔还让人想起后来的现象学家，包括海德格尔、萨特和梅洛庞蒂，胡塞尔在这个问题上与他们都颇有相似之处。
